## ğŸ¯ Key Decision Points

### 1. Browser Detection
- **Input:** Process path from active window
- **Process:** Pattern matching against config
- **Output:** Browser configuration or null

### 2. Configuration Source
- **Priority 1:** Backend API
- **Priority 2:** Local Cache
- **Priority 3:** Default Hardcoded

### 3. URL Extraction Method
- **Windows 10/11:** Native tool â†’ History fallback
- **Other Windows:** History only
- **Linux:** History only
- **macOS:** Basic info only

### 4. History File Location
- **Windows:** From Local State file (Chrome-based) or config path
- **Linux/Mac:** Recursive search using config file name

### 5. SQL Query Selection
- **Source:** Browser config from backend
- **Type:** Based on browser (chromium, firefox, custom)
- **Fallback:** Default query if config missing
